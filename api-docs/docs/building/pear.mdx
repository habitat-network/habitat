# Pear (permission-enforcing AT Protocol repository)

Habitat previously designed and implemented a protocol for permissioned data on AT Protocol (called cliques). As the ecosystem is converging on the Permissioned Spaces proposal, we are deprecating cliques, and migrating all permissioned data to the spaces proposal. Products seeking to use private data on the protocol with spaces can use our API today, and we plan to migrate these to users' PDS'es once spaces is emrged and widely supported. At that point, pear will be fully deprecated, and Habitat will only support Organizational Data Servers.
