# What is habitat?

Habitat is an extension of ATProtocol that makes it possible for users to have full transparency and control into their personal data, and developers to build social apps for the internet without storing any user data. Our pear node (short for permission-enforcing ATProtocol repository) mirrors a generic public ATProtocol repository and its APIs, but gives users and applications fine-grained control over record-level permissions. The sweet spot for building on habitat is creating local-first applications that operate on private data and can inter-operate with other applications in the ATProtocol ecosystem.
