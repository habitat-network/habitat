# Habitat API

Welcome to the Habitat API reference! We are currently under development and our API may make breaking changes without notice. Stay tuned [@habitat.network](https://bsky.app/profile/habitat.network) on Bluesky and [habitat.leaflet.pub](https://habitat.leaflet.pub) for future formal releases.

## Endpoints

All endpoints are served over HTTPS at:

```
https://habitat.network/xrpc
```

## Authentication

Authenticated endpoints use HTTP Bearer auth. Pass your token in the `Authorization` header:

```
Authorization: Bearer <token>
```
